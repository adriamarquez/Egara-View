Què és Egara-View?

Egara-View es una plataforma implementada i desenvolupada amb el llenguatge Python que, mitjançant un cercador i classificador visual, reconeix els diferents edificis emblemàtics de Terrassa.

Els usuaris poden prendre una fotografia d'aquests edificis i obtenir informació d'interés sobre aquest edifici si es troba dins la nostra base de dades.

Com funciona?

L'aplicació en la que estem treballant, tindrà la funció de detectar els punts característics de les imatges que li passem (com en la imatge d'exemple de més abaix) i comparar-les amb una base de dades per determinar de quin lloc es tracta. Això ho aconseguim entrenant el nostre software amb la base de dades de la que disposem, de forma que aprengui les diferents característiques de cada atracció turística.

Inicialment, ens centrarem en dotze edificis diferents de Terrassa, però aquest nombre de espais turístics es podría incrementar.
