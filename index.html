Què és Egara-View?

Egara-View es una plataforma implementada i desenvolupada amb el llenguatge Python que, mitjançant un cercador i classificador visual, reconeix els diferents edificis emblemàtics de Terrassa.

Els usuaris poden prendre una fotografia d'aquests edificis i obtenir informació d'interés sobre aquest edifici si es troba dins la nostra base de dades.

Per què Egara-View?

Egara-View ha estat disenyat com a projecte de l'assignatura 'Gestió i distribució de senyals audiovisuals'. Es pot fer servir per qualsevol dubte relacionat amb edificis que desconeixes sobre Terrassa amb el simple fet de fer-ne una fotografaia.

Com funciona?

Egara-View té la funció de detectar i cercar els punts característics de la imatge (o de les imatges) desitjades i comparar-les amb moltes fotos d'una base de dades per determinar de quin edifici es tracta. Això ho aconseguim entrenant el nostre software amb la base de dades d'entrenament de la que disposem, de forma que aprengui les diferents característiques de cada atracció turística i calculi un rànquing amb les imatges que s'assemblen més.
